Common questions

Answers before you start

What projects do you handle?

Kitchen, bath, and living space remodeling are the core service paths

Do I need drawings first?

No. Photos, notes, rough measurements, or a simple description are enough to begin

What should I send first?

Send the room, goals, location, timing, and any photos that show the current space

Can I send photos by WhatsApp?

Yes. Photos are often the quickest way to understand condition, scope, and direction

When do we discuss price?

Price makes sense after the basic scope is clear enough to review responsibly

What changes price the most?

Layout changes, plumbing, electrical, materials, structure, and existing conditions

How long does a project take?

Timing depends on scope, materials, permits, scheduling, and site conditions

Do permits matter?

Permit needs depend on the project location, project scope, and local code rules

How are changes handled?

Changes should be reviewed before work moves so cost, timing, and materials stay clear
